# GymNerd
A PWA (Progressive Web App) to track your performance at the gym.

## Features
- Routine creation
- Play a routine live (with timer and set/reps/weight info)
- Add ah-hoc history of routines
- Add and track body weight
- Track progress in the statistics page
- Gamified progress with streaks and records
- Syncronization with Google Drive (where the data is stored for use across devices/platforms)
